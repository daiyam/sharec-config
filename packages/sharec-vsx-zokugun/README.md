Getting Started
---------------

```sh
mkdir myproject
cd myproject
npm init -y
npm i -D @daiyam/sharec-base @daiyam/sharec-lang-ts @daiyam/sharec-vsx-ts @daiyam/sharec-vsx-zokugun
```
